# Stratégie d'Internationalisation - TaxySoa

## 1. Langues Supportées

### Priorité des Langues
1. **Français** (fr) - Langue principale
   - Interface complète
   - Documentation
   - Support client

2. **Malgache** (mg) - Langue locale
   - Interface complète
   - Support client prioritaire
   - Adaptation culturelle

3. **Anglais** (en) - Langue secondaire
   - Interface complète
   - Documentation technique

## 3. Convention de Nommage

### Format : `[domaine].[contexte].[action]`

Exemples :
```php
// Authentification
'auth.login.title' => 'Connexion'
'auth.register.submit' => 'S\'inscrire'

// Réservation
'booking.steps.pickup' => 'Point de départ'
'booking.steps.destination' => 'Destination'

// Messages système
'system.notification.booking_accepted' => 'Votre course a été acceptée'
```

### Configuration Vue I18n
```javascript
// resources/js/i18n/index.js
import { createI18n } from 'vue-i18n'
import fr from './fr'
import mg from './mg'
import en from './en'

export default createI18n({
    locale: localStorage.getItem('locale') || 'fr',
    fallbackLocale: 'fr',
    messages: { fr, mg, en }
})
```

## 6. Sélecteur de Langue

### Composant Vue.js
```vue
<!-- resources/js/components/i18n/LanguageSelector.vue -->
<template>
    <div class="language-selector">
        <button 
            v-for="lang in languages" 
            :key="lang.code"
            :class="{ active: currentLocale === lang.code }"
            @click="changeLanguage(lang.code)"
        >
            {{ lang.name }}
        </button>
    </div>
</template>

<script>
export default {
    data() {
        return {
            languages: [
                { code: 'fr', name: 'Français' },
                { code: 'mg', name: 'Malagasy' },
                { code: 'en', name: 'English' }
            ]
        }
    },
    computed: {
        currentLocale() {
            return this.$i18n.locale
        }
    },
    methods: {
        changeLanguage(locale) {
            this.$i18n.locale = locale
            localStorage.setItem('locale', locale)
            document.documentElement.lang = locale
        }
    }
}
</script>

<style scoped>
.language-selector {
    display: flex;
    gap: 8px;
}

.language-selector button {
    padding: 8px 16px;
    border: 1px solid var(--neutral-gray);
    border-radius: 4px;
    background: var(--white);
    cursor: pointer;
}

.language-selector button.active {
    background: var(--secondary-blue);
    color: var(--white);
    border-color: var(--secondary-blue);
}
</style>
```

## 7. Bonnes Pratiques

1. **Traductions**
   - Utiliser des clés descriptives
   - Éviter les traductions littérales
   - Maintenir un glossaire commun

2. **Formatage**
   - Utiliser les helpers Laravel
   - Adapter les formats selon la locale
   - Tester les formats sur différentes tailles

3. **Performance**
   - Charger les traductions de manière asynchrone
   - Mettre en cache les traductions
   - Optimiser le bundle de traduction

4. **Tests**
   - Tester toutes les langues
   - Vérifier les formats de date/heure
   - Tester le RTL si nécessaire
